# Automated-Super-Store-Sales-Dashboard
Automated Sales Dashboard using Excel, Power BI and Power Automate Desktop with workflow automation.


## Project Overview

This project demonstrates an automated reporting workflow using Excel, Power BI Desktop, and Power Automate Desktop.
The automation flow opens the data file, launches the Power BI dashboard, and updates the report automatically.

## Tools Used

* Microsoft Excel
* Power BI Desktop
* Power Automate Desktop

## Workflow

1. Sales data stored in Excel / CSV file
2. Power Automate Desktop runs the automation flow
3. Excel data file is opened
4. Power BI dashboard file is launched
5. Dashboard reads updated data
6. Updated dashboard is displayed automatically


## Dashboard Visuals

* Total Sales KPI
* Total Orders KPI
* Sales by Category
* Sales by Sub-Category
* Sales by Ship Mode
* Monthly Sales Trend
* Region Filter

## Use Case

This project simulates a real-world reporting workflow used by Data Analysts to automate dashboard updates.
